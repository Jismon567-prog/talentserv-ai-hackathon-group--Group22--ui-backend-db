**Project Name**: OpenMRS AI Healthcare Test Automation Agent
**Goal**: Build an intelligent AI agent that takes healthcare requirements/user stories and generates:
- High-quality test cases (Functional, Negative, Validation, Security, Privacy, Audit)
- Synthetic test data (Patients, Visits, Encounters)
- Automation skeletons (Playwright + API)
- Coverage report + Safety checklist

**Core Rules**:
- Must reference OpenMRS concepts: Patient, Visit, Encounter, Obs, PatientIdentifier, User, Role, Privilege
- All patient data must be 100% synthetic/fake
- Show clear multi-stage agent workflow
- Use TypeScript, Tailwind, shadcn/ui style
- Third-party auth using Clerk
