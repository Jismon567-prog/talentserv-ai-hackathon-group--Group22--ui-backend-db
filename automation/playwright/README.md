# OpenMRS Agent — Playwright Automation Skeleton

Runnable smoke tests aligned with the **Playwright + REST** snippets generated by Stage 5 (`lib/automation-templates.ts`).

## Quick run

```bash
cd automation/playwright
npm install
npx playwright install chromium
npm test
```

## Point at OpenMRS

```bash
export BASE_URL=https://your-openmrs-server.org
npm test
```

## What this proves

| Test | Pattern |
|------|---------|
| `TC-SMOKE-001` | UI navigation (`page.goto`, URL assertion) |
| `TC-SMOKE-002` | REST call with Basic auth header + status tolerance |

Default `BASE_URL` is `https://example.com` so **CI and judges** can run tests without a live OpenMRS instance.

## CI

GitHub Actions job `playwright-smoke` runs these tests on every push/PR — see `.github/workflows/ci.yml`.
